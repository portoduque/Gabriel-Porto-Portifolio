Tarefas Desenvolvidas no portal do CIGES

Implementação da Interface do Usuário (UI)

  • Desenvolvi layouts responsivos e intuitivos utilizando HTML5, CSS3 e JavaScript, alinhados à estrutura semântica recomendada pelo MDN.
  • Criação de dashboards interativos para visualização de indicadores de saúde pública, com gráficos dinâmicos e filtros de dados personalizados via JavaScript Vanilla.
  • Garantia de acessibilidade e legibilidade para gestores de saúde, seguindo diretrizes do W3C para interfaces inclusivas.

Otimização para Dispositivos Móveis

  • Uso de media queries, CSS Grid/Flexbox e Bootstrap para garantir compatibilidade em smartphones e tablets, priorizando a usabilidade em ambientes corporativos.
  • Testes de responsividade em navegadores Chrome, Firefox e Edge, assegurando experiência consistente para usuários.

Colaboração com a Equipe Multidisciplinar

  • Integração com APIs REST da equipe backend da SETIC - RO, priorizando segurança e eficiência na troca de dados estratégicos de saúde
  • Participação em reuniões de alinhamento para validar requisitos técnicos e institucionais, garantindo conformidade com as diretrizes da SESAU.
  • Trabalho conjunto com designers para padronizar estilos e garantir interface alinhada às normas de governança digital do portal.

Aprendizados e Experiência Profissional Adquiridos

  • Domínio avançado de Bootstrap, JavaScript e técnicas de otimização de performance web para plataformas de alto impacto social.
  • Resolução de desafios técnicos, como ajuste de layouts para navegadores antigos e integração segura de APIs REST com autenticação OAuth.
  • Gerenciamento de prazos rigorosos em projeto crítico, equilibrando qualidade técnica e entregas ágeis sob pressão.

Habilidades Técnicas Destacadas

  • Frontend (HTML5, CSS3, JavaScript)
  • Responsive Design e Acessibilidade
  • Otimização de Performance
  • Integração de APIs REST
  • Trabalho em Equipe Multidisciplinar
  • Soluções para Governança Digital e Saúde Pública

Resultado

Um portal moderno, acessível e funcional, que centraliza dados críticos de saúde pública, facilitando decisões baseadas em evidências para a SESAU em Rondônia. A plataforma permite análise de tendências, monitoramento de indicadores e comparações regionais, apoiando gestores na alocação eficiente de recursos

Descrição

  • ciges.sesau.ro.gov.br

O site Painel CIGES é uma plataforma digital que centraliza dados estratégicos de saúde para apoiar a gestão do Sistema Único de Saúde (SUS) em Rondônia. Ele oferece relatórios detalhados, indicadores de saúde, análises estratégicas e ferramentas de visualização de dados, permitindo que gestores, pesquisadores e profissionais de saúde tomem decisões baseadas em evidências
